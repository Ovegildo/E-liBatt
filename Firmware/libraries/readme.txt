Configurar el IDE de Arduino para operar desde la carpeta Firmware.

Ir a Archivo > Preferencias > Ubicacioón del Sketchbook > Navegar

Seleccionar la carpeta Firmware del proyecto en github, de modo que quede similar a:

   C:\Users\rene\Desktop\Archive\work\Elibatt\github\E-liBatt\Firmware

Reiniciar Arduino, y corroborar que Sketch > Importar Librería... ofrezca las nuevas librerias.

Luego abrir algún .ino de firmware (Master o Slave) y presionar el botón Cargar.
